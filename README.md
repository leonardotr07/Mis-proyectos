Descripción sobre el software:
El sistema web “WearDrop” propuesto se centra en la automatización de los procesos transaccionales
que sostienen la operación diaria de una tienda de ropa en Gamarra. En primer lugar, permite registrar y
gestionar las ventas, desde la selección de productos hasta la emisión de comprobantes de pago electrónicos
(boletas, facturas, notas de crédito y débito). 

Cada transacción de venta actualiza automáticamente el inventario, genera el detalle de productos adquiridos, 
registra el método de pago y consolida la información del cliente para fines de fidelización. 
Del mismo modo, el sistema incorpora la gestión de compras a proveedores, donde se registran los lotes 
de mercadería ingresados, los montos pagados y las deudas pendientes, asegurando un control transparente
del abastecimiento y de las obligaciones financieras de la tienda.

Adicionalmente, el software gestiona el control de inventarios en tiempo real, actualizando entradas
y salidas de mercadería, incluyendo devoluciones tanto de clientes como hacia proveedores en caso de productos defectuosos.
Estas operaciones están vinculadas con un módulo de almacenes y lotes, que permite
organizar los productos según su procedencia, fecha de ingreso y estado. Complementariamente, el sistema
genera reportes automáticos que consolidan la información transaccional: estadísticas de ventas diarias,
semanales o mensuales; márgenes de ganancia bruta por línea de producto; y listados de productos más y
menos vendidos.

En conjunto, estas funciones transaccionales otorgan al negocio un flujo de información
preciso y en tiempo real, reduciendo riesgos de errores, evitando quiebres de stock y fortaleciendo la toma
de decisiones estratégicas.
